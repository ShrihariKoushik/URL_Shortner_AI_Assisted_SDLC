Run Locally

cd ai_assisted_shortener
python -m pip install -r requirements.txt
python -m uvicorn app.main:app --host 127.0.0.1 --port 8010

Open http://127.0.0.1:8010/ for the UI and http://127.0.0.1:8010/docs for API docs.
